## 三、动作说明

### 1）ip_in_range
动作描述：判断IP地址是否在某个段内
对应接口：

**入参说明**

| 参数  | 类型 | 数据样例 | 必须 | 默认值 | 说明   |
| ---- | ----- | ---- | ---- | ---- | ---- |
| ip |  string  |  | True |  |  要判断的IP地址，Demo: 192.168.1.123  |
| start |  string  |  | True |  |  起始IP地址， Demo: 192.168.1.1  |
| end |  string  |  | True |  |  结束IP地址， Demo: 192.168.1.254  |

**出参说明**    

| 参数  | 类型   | 数据样例  | 说明 |
| ---- | ----- | ------- | ---- |  
| action_result.is_range |  boolean  |  |  如果在段内则返回真，否则返回假  |
