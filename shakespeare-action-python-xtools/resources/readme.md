## 二、app使用注意事项

1）RSA加密部分

- 不建议敏感系统直接使用应用动作，避免密钥泄露
- JS代码的填充方式默认是：PKCS1v15
- 通过OpenSSL生成的证书文件，可能有密码保护


2）依赖项

```
pip install cryptography
```
